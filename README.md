# Vibecoding 🚀

> 个人工作效率追踪网站 — 记录工作成果，可视化效率趋势

## 技术栈

- **Framework**: Next.js 16 (App Router)
- **Language**: TypeScript
- **Styling**: Tailwind CSS v4
- **Charts**: Recharts
- **Storage**: Browser localStorage
- **Deploy**: Vercel

## 功能

- 📝 **记录工作** — 输入项目、时长、类型、标签、反思
- 📊 **仪表盘** — 今日/本周/本月统计，趋势图，类型分布
- 🏷️ **标签系统** — 灵活标记工作内容
- 🌙 **深色模式** — 自动跟随系统主题
- 📱 **响应式** — 移动端友好

## 快速开始

```bash
git clone https://github.com/JTCAO515/vibecoding.git
cd vibecoding
npm install
npm run dev
```
